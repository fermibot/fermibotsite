Geometry | Rotating Squares


Objective

This interactive visualization recreates the Mathematica rotating squares pattern using D3.js. The original code generates a series of squares with increasing sizes, where each square is rotated by an angle proportional to the square of its radius.

Original Mathematica Code:
Graphics[
 Table[Rotate[{Opacity[0.017], Pink,
    EdgeForm[{Thin,
      RandomChoice[{Green, Darker[Green], Lighter[Blue]}],
      Opacity[0.15]}], Rectangle[{-r, -r}, {r, r}]}, r^2], {r, π,
   10 π, π/13}]]

How It Works

This visualization replicates the Mathematica code exactly using D3.js:

  1. Generates squares with radii from \(\pi\) to \(10 \pi\) in steps of \(\pi/13\)
  2. Applies rotation to each square based on the formula: \(R = r^2 f \mid\) \(R \rightarrow\) Rotation, \(r \rightarrow\) radius, \(f \rightarrow\) Rotation factor.
  3. Sets square fill opacity to 0.017 (almost transparent)
  4. Sets border opacity to 0.15 with thin strokes
  5. Randomly selects border colors from {Green, Dark Green, Light Blue}
  6. Uses pink (#ff69b4) as the fill color

The interactive controls allow you to modify each parameter and see the effect in real-time.

Technical Implementation

The visualization is implemented using D3.js, which provides:

  • SVG Rendering: All squares are rendered as SVG rectangles
  • Zoom & Pan: Interactive navigation using D3 zoom behavior
  • Real-time Updates: Parameter changes update the visualization without page reload
  • Tooltips: Hover information shows detailed square properties
  • Animation: Continuous rotation can be enabled/disabled with speed control
