Home Temperature Dynamics

Problem Statement

You turn off the heat in your home two hours before going to bed. The temperature in your home follows Newton's Law of Cooling, which states that the rate of temperature change is proportional to the difference between the current temperature and the ambient (outside) temperature.

Given:

  • Temperature at \(t = 0\) (when heat is turned off): \(T(0) = 66°\text{F}\)
  • Temperature at \(t = 2\) hours (bedtime): \(T(2) = 63°\text{F}\)
  • Outside temperature (constant): \(T_A = 32°\text{F}\)

Find: The temperature at \(t = 10\) hours (8 hours after bedtime).

Analytical Solution

Newton's Law of Cooling:

The rate of temperature change is proportional to the difference between current temperature \(T\) and ambient temperature \(T_A\):

\[\frac{dT}{dt} = k(T - T_A) = k(T - 32)\]

General Solution:

  • This is a first-order linear ODE with separable variables
  • \(\frac{dT}{T - 32} = k \, dt\)
  • \(\int \frac{dT}{T - 32} = \int k \, dt\)
  • \(\ln(T - 32) = kt + C_1\)
  • \(T - 32 = C e^{kt}\)
  • \(T(t) = 32 + C e^{kt}\)

Applying Boundary Conditions:

  • From \(T(0) = 66\):
    • \(66 = 32 + C e^{k \cdot 0}\)
    • \(C = 34\)
  • From \(T(2) = 63\):
    • \(63 = 32 + 34 e^{2k}\)
    • \(31 = 34 e^{2k}\)
    • \(e^{2k} = \frac{31}{34}\)
    • \(2k = \ln\left(\frac{31}{34}\right)\)
    • \(k = \frac{1}{2}\ln\left(\frac{31}{34}\right) \approx -0.046187\)

Specific Solution:

\[T(t) = 32 + 34 e^{-0.046187t}\]

Temperature at t = 10:

  • \(T(10) = 32 + 34 e^{-0.046187 \times 10}\)
  • \(T(10) = 32 + 34 e^{-0.46187}\)
  • \(T(10) = 32 + 34 \times 0.6303\)
  • \(T(10) \approx 32 + 21.43\)
  • \(T(10) \approx 53.43°\text{F}\)

Physical Interpretation
  • Exponential Decay: The temperature decreases exponentially toward the ambient temperature
  • Equilibrium: As \(t \to \infty\), \(T(t) \to T_A = 32°\text{F}\)
  • Cooling Rate: The constant \(k \approx -0.046\) indicates the rate of heat loss
  • Half-life Concept: The time for the temperature difference to halve can be calculated as \(t_{1/2} = \frac{\ln(2)}{|k|} \approx 15\) hours
Mathematica Code

Below is the Mathematica code used to solve this differential equation and generate parametric studies:


(* Define the differential equation *)
ode = D[T[t], t] == k (T[t] - TA);

(* Solve with boundary conditions *)
solution = DSolve[{ode, T[0] == 66, T[2] == 63}, T[t], t, {k, TA}];

(* Specific solution with TA = 32 *)
specificSolution = DSolve[{
    D[T[t], t] == k (T[t] - 32),
    T[0] == 66,
    T[2] == 63
}, T[t], t];

(* Extract the solution *)
T[t_] = T[t] /. specificSolution[[1]];

(* Evaluate at t = 10 *)
T[10]

(* Plot the temperature curve *)
Plot[T[t], {t, 0, 24},
    PlotRange -> All,
    Frame -> True,
    FrameLabel -> {"Time (hours)", "Temperature (°F)"},
    PlotLabel -> "Home Temperature vs Time",
    GridLines -> Automatic,
    ImageSize -> Large
]

(* Parametric Study 1: Varying Outside Temperature *)
Module[{solutions, temps},
    temps = Range[20, 40, 5];
    solutions = Table[
        DSolve[{
            D[T[t], t] == k (T[t] - TA),
            T[0] == 66,
            T[2] == 63
        }, T[t], t] /. {TA -> temp},
        {temp, temps}
    ];

    Plot[Evaluate[T[t] /. solutions], {t, 0, 24},
        PlotRange -> All,
        Frame -> True,
        PlotLegends -> ("TA = " <> ToString[#] <> "°F" & /@ temps),
        PlotLabel -> "Effect of Outside Temperature",
        ImageSize -> Large
    ]
]

(* Parametric Study 2: Varying T(2) *)
Module[{solutions, t2temps},
    t2temps = Range[58, 66, 2];
    solutions = Table[
        DSolve[{
            D[T[t], t] == k (T[t] - 32),
            T[0] == 66,
            T[2] == t2
        }, T[t], t],
        {t2, t2temps}
    ];

    Plot[Evaluate[T[t] /. solutions], {t, 0, 24},
        PlotRange -> All,
        Frame -> True,
        PlotLegends -> ("T(2) = " <> ToString[#] <> "°F" & /@ t2temps),
        PlotLabel -> "Effect of Bedtime Temperature",
        ImageSize -> Large
    ]
]
Key Insights
  • The temperature drops from 66°F to approximately 53.43°F over 10 hours
  • The rate of cooling slows down as the temperature approaches the outside temperature
  • Newton's Law of Cooling assumes constant outside temperature and no heat sources
  • The negative value of \(k\) indicates cooling; positive would indicate heating
  • This model is widely applicable: coffee cooling, crime scene temperature analysis, etc.
