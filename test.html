JavaScript Code:
// Function to apply styles to an element by ID
function applyStyles() {
    // Get the element by its ID
    const element = document.getElementById('exampleImage');

    // Apply the styles
    element.style.width = '200px';
    element.style.height = '200px';
    element.style.objectFit = 'cover';
    element.style.borderRadius = '50%';
    element.style.border = '3px solid #4361ee';
    element.style.background = 'white';
    element.style.padding = '5px';
    element.style.boxShadow = '0 5px 15px rgba(0, 0, 0, 0.1)';
    element.style.display = 'block';
    element.style.marginLeft = 'auto';
    element.style.marginRight = 'auto';
}

// Function to reset styles
function resetStyles() {
    const element = document.getElementById('exampleImage');

    // Reset styles to initial values
    element.style = '';
}

How This Works:

  • ID Selection: We use document.getElementById('exampleImage') to select the element
  • Style Property Access: We access the style property of the element
  • CSS Property Assignment: We assign values to specific CSS properties using camelCase notation (e.g., borderRadius instead of border-radius)
  • Direct Manipulation: Styles are applied directly to the element's inline style attribute
  • Reset Functionality: Setting element.style = '' removes all inline styles
